# ROS Autonomy Simulation

## Clone


```bash
cd ~/catkin_ws/src
```
```bash
git clone https://github.com/SinLoops333/4WD_description.git
cd ~/catkin_ws 
```

```bash
catkin build 
```

```bash
source ./devel/setup.bash
source ~/.bashrc 
```

## Launch Rviz and Gazebo
```bash
roslaunch 4WD_description gazebo.launch 
```
